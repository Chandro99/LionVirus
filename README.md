## Deployment on Termux

To deploy this on termx

```bash
  apt update
  apt upgrade
  git clone https://github.com/Chandro99/RClogger
  cd RClogger
  bash recommended.sh
  python RClogger.py
```

## Deployment on Kali Linux
To deploy in kali 
```bash
  apt-get update
  apt-get upgrade
  git clone https://github.com/Chandro99/RClogger
  cd RClogger
  bash recommended.sh
  python RClogger.py
```
